Recipe 三配置同 pipeline：per-file z-score 標準化 + 單一閾值 sweep A8 alone SD alone 4-vec ENS

# Per 0.5s segment, three configs of combo: A8_alone = zscore(cos_z + asnorm_z + discr_z) # thr = -0.15 SD_alone = zscore(mean over seg of (p[1] − (p[0]+p[2]+p[3]))) # thr = -0.05 ENS_4vec = 0.5·cos_z + 0.5·discr_z + 2.0·sd_z # thr = 0.05 # pred = combo ≥ thr, expand to per-0.1s segments for metric

A8/SD/ENS 三者**統一在 per-file z-score + 閾值 sweep 同 framework** 比較，不是各自原始輸出規則。SD-alone 採連續分數，不是二元規則 p[1] > p[0]+p[2]+p[3]（後者 F1=0.84，較弱）。
